# wyqd_demo

审计大模型系统前端 Demo，当前重点页面为审计工作台。

## 本地运行

```powershell
npm install
npm run dev
```

默认开发地址为 `http://127.0.0.1:5173`，如果端口被占用，Vite 会自动切换到可用端口。

## 验证

```powershell
npm run build
npm run check
```

## 当前工作台能力

- 有数据工作台：指标卡、我的待办、最近审计任务、任务进度概览、风险提醒、最近生成结果、快捷入口、系统通知、最近操作日志。
- 空白工作台：关键指标为 0，新手引导、流程预览、创建首个审计任务、导入模拟数据和查看演示流程入口。
- 页面右上角支持在“查看空白页”和“导入模拟数据”之间切换展示状态。
