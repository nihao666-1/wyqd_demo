## 自适应策略

- 1600px 左右：保持三列整体结构，页面接近截图比例。
- 1366/1440px：仍保持左导航 + 中主列 + 右侧栏，主列内部卡片和图表压缩，避免横向滚动。
- 1024px 左右：右侧栏下移，两张能力卡可继续双列，统计卡降为 3+2 或自适应网格。
- 768px 以下：主列、右栏、能力卡、图表、统计卡全部单列；表格保留横向滚动。
- 图表用 `viewBox` SVG 和 CSS 网格自适应，避免固定像素导致窗口缩放时比例失衡。
