# 配置中心三子页面原型任务书

## 目标
- 仅修改配置中心页面，保持其他页面不变。
- 基于三张效果图实现三个可切换原型：配置中心初始页、配置与记录中心、系统参数配置页。
- 顶部提供状态切换按钮，便于在同一路由 `/config` 查看三个状态。
- 页面布局要贴近效果图红框区域：顶部 Tab、左侧分类、中间主内容、右侧留痕/检查区。
- 文字边距不能太紧凑，避免截断、重叠、大面积空白。

## Agent 分工提示词

### Agent A：配置中心初始页拆解
提示词：只分析“配置中心初始页”。输出红框区域拆解、必须出现的文字和数据、布局比例、容易出错点。重点覆盖指标卡、暂无用户自定义配置、内置配置列表、初始化检查、下一步建议。

### Agent B：配置与记录中心拆解
提示词：只分析“配置与记录中心有数据页”。输出顶部横向 Tab、左侧配置分类、中间表格、右侧变更与留痕的内容结构、表格列、按钮、状态标签、分页和响应式风险。

### Agent C：系统参数配置页拆解
提示词：只分析“系统参数配置页”。输出参数 Tab、左侧参数分类、中间模型参数表单与参数变更预览、右侧参数变更与留痕内容、必备字段、行距边距和底部对齐风险。

## 区域拆解

### 页面一：配置中心初始页
- 顶部标题：配置中心，描述集中管理模板、规则、标签、权限与系统参数。
- 顶部 Tab：模板配置、规则配置、标签配置、权限配置、系统参数配置。
- 指标卡：内置模板 6、用户模板 0、启用规则 0、待确认规则 4、参数变更 0。
- 左侧配置分类：报告模板配置、审计规范模板配置、费用异常规则配置、制度比对规则配置、文件标签配置、导出模板配置、AI 生成标识配置、多模型复核配置、系统参数配置。
- 中间空态：暂无用户自定义配置。说明系统已预置基础模板和规则，请确认适用范围后启用，也可新增模板、规则和参数版本。
- 中间下方内置配置列表：配置名称、配置类型、版本号、状态、适用模块、最近修改人、修改时间、操作。
- 右侧初始化检查：报告模板、费用规则、标签字典、权限角色、AI 标识。
- 右侧下一步建议：确认报告模板、启用费用规则、设置 AI 生成标识、创建首个任务。

### 页面二：配置与记录中心
- 顶部标题：配置与记录中心。
- 顶部横向 Tab：模板配置、规则配置、标签配置、权限配置、版本记录、操作留痕、导出记录、复核记录。
- 左侧配置分类：报告模板配置、审计规范模板配置、费用异常规则配置、制度比对规则配置、文件标签配置、导出模板配置、AI 生成标识配置、多模型复核配置。
- 中间操作条：新增规则、复制版本、规则试运行、状态筛选、规则名称搜索、重置。
- 中间表格列：规则名称、适用模块、版本号、状态、命中样例、最近修改人、修改时间、操作。
- 操作项：查看、复制、停用/启用、日志。
- 右侧变更与留痕：版本历史、操作日志、复核意见、导出记录。

### 页面三：系统参数配置页
- 顶部标题：系统参数配置。
- 顶部操作：草稿待提交、保存草稿、参数试运行、提交复核、发布生效。
- 参数 Tab：基础参数、文件参数、模型参数、AI 标识、安全参数、通知参数、导出参数、审计留痕。
- 左侧参数分类：模型调用参数、多模型复核参数、敏感词参数、生成失败重试、任务超时控制、版本保存策略、导出水印参数、日志保留周期。
- 中间模型参数表单：默认模型、高风险复核模型、最大上下文长度、单次生成超时时间、失败重试次数、生成结果保留天数、多模型复核、AI 生成标识、敏感词处理方式。
- 中间参数变更预览：参数项、当前值、修改后、生效范围、是否需复核。
- 右侧参数变更与留痕：当前版本、版本历史、变更摘要、影响模块、操作日志、复核意见、提交复核按钮。
