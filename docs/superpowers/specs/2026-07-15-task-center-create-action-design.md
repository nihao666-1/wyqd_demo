# 任务中心创建入口设计

## 目标

将任务中心中独占一行的深红色“创建审计任务”按钮，改为与五张任务指标卡同排的第六个轻量操作卡，消除按钮悬空和额外纵向留白。

## 设计

- 删除只承载创建按钮的 `.task-center-header`。
- 在 `.metric-strip` 尾部放置链接卡，文案保持“创建审计任务”，继续跳转 `/tasks/create`。
- 操作卡与指标卡等高、同圆角、白色背景；使用审计红细边框、红色创建图标和红色文字，不使用大面积深红填充或阴影。
- 宽屏六列同排；现有 `1200px`、`760px` 断点继续分别降为三列、两列，不新增响应式体系。
- 不调整指标数据、筛选区、任务表格、右侧栏或创建任务流程。

## 验收

- 数据态任务中心不再渲染 `.task-center-header`。
- `.metric-strip` 内依次包含五个指标按钮和一个 `/tasks/create` 操作卡。
- 桌面默认网格为六列，操作卡具有轻量红色描边样式。
- 只运行任务中心组件布局相关测试，不运行全量测试。
