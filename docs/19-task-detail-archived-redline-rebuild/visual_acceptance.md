## 响应式规则

- 画布：`1600×995`；宽度缩放 `clamp(0.85, viewportWidth / 1600, 1)`。
- 1366×768：约 0.854，开启紧凑高度、主区/表格内部滚动、右栏内容滚动 + 底部操作 sticky。
- 1440×900：0.9，完整首屏为主。
- 1600×995：1.0，区域坐标误差目标 ±3px。
- 1920×1080：不放大，内容受控居中，额外高度不用于拉大行高。
- 所有视口：`documentElement.scrollWidth <= clientWidth + 1`，右栏不下沉、不重叠。
