# 报告智能化页面重构任务书

## 目标

只修改 `src/views/audit-report/AuditReportWorkbench.vue`，保留 `/audit-report/workbench` 路由和审计报告资料导入入口 `/audit-report/material/import`。在同一个页面内提供三个子页面切换：报告智能化空白页、报告生成页、报告审核页。页面要严格表达报告智能化包含“报告生成”和“报告审核”，并覆盖正文编辑、依据核对、章节重生成、审核定位、问题处理、复核记录和导出留痕。

## 红框区域拆解

### 子页面一：报告智能化空白页

1. 顶部指标区
   - 标题：报告智能化。
   - 指标：报告任务、草稿报告、待审核报告、审核问题、导出文件。
   - 每个指标显示数量 0 和较昨日 0。

2. 空任务引导区
   - 标题：暂无报告任务。
   - 文案说明可根据审计任务、审计问题、风险事项和模板生成报告，也可上传报告进行格式与文字审核。

3. 报告生成入口
   - 流程：选择任务、选择报告模板、引用结果、生成草稿。
   - 主按钮：新建报告生成。

4. 报告审核入口
   - 流程：上传报告、选择审核规则、执行检查、形成问题清单。
   - 主按钮：上传报告审核。

5. 报告列表
   - 表头：报告编号、报告名称、报告类型、版本、状态、创建人、更新时间、操作。
   - 空状态：暂无数据。
   - 筛选控件：报告类型、日期范围、报告名称、筛选、刷新。

6. 报告流程
   - 节点：资料预检、生成章节、插入依据、人工编辑、智能审核、导出记录。

7. 右侧报告类型与模板
   - 报告类型：营业部常规审计报告、营业部反洗钱审计报告、营业部负责人离任审计报告。
   - 格式模板：Word 格式模板、PDF 格式模板、Excel 格式模板、模板示例库。
   - 按钮：管理模板与格式。

### 子页面二：报告生成页

1. 顶部配置条
   - 审计任务、报告类型、报告模板、输出格式。
   - 开关：AI 标识、多模型复核。
   - 操作：开始生成、章节级重新生成、保存版本、提交审核、导出报告。

2. 左侧章节目录
   - 搜索章节。
   - 章节：基本情况、审计发现的主要问题、审计意见及整改要求、附件材料。
   - 子章节可点击切换。
   - 按钮：新增章节。

3. 中间报告编辑预览
   - 工具栏：正文、字体、字号、加粗、下划线、颜色、对齐、列表、链接、图片、表格、格式检查。
   - 正文展示 AI 段落、待优化段落、问题段落。
   - 标识：依据、问题、AI。
   - 支持章节级重新生成和人工编辑入口。

4. 右侧依据来源
   - 分类：引用结果、制度条款、费用异常、监督共享资料、审核建议。
   - 每条依据显示来源、匹配度、查看原文/查看详情。
   - 底部：只显示高匹配度、刷新。

5. 底部生成流程
   - 读取资料、生成章节、插入依据、人工编辑、智能审核、导出记录。
   - 每步显示状态和数量。

6. 右下生成建议
   - 当前生成进度 66%。
   - 预计剩余时间 6 分钟。

### 子页面三：报告审核页

1. 顶部标题与操作
   - 标题：报告审核。
   - 操作：返回任务、上传新版本、重新检查、生成复核记录、导出问题清单。

2. 基础信息
   - 审计单位、审计期间、报告类型、检查规则、状态、检查完成时间。

3. 左侧上传与规则选择
   - 已上传文件、报告类型、模板版本、检查规则、检查范围。

4. 中间审核进度与问题统计
   - 上传报告、解析文档、格式检查、内容优化、人工处理、形成复核记录。
   - 问题统计：问题总数、高风险、中风险、低风险、已采纳建议。

5. 问题清单
   - Tab：全部问题、格式问题、字号字号、标题编号、表格问题、依据完整性、文字优化。
   - 表格字段：问题编号、问题类型、位置、问题描述、风险等级、建议处理、状态、操作。
   - 操作：定位、采纳建议、忽略、人工修改。

6. 预览区
   - 上传检查摘要。
   - 报告定位预览，问题编号在正文中标注。

7. 右侧问题详情与处理
   - Tab：基本信息、规则依据、修改建议、修改前后对比、处理记录。
   - 基本信息字段：问题编号、问题类型、位置、风险等级、检查规则、问题描述、综合建议。
   - 处理意见：采纳建议、忽略建议、标记已修复。
   - 处理说明输入框。
   - 操作记录表格。
   - 底部按钮：采纳建议、忽略建议、标记已修复、写入复核记录。

## Agent 分工

### Agent A：现有实现定位

提示词：只读分析报告智能化相关页面。定位路由、页面文件、共享组件/样式，输出当前实现结构、最小修改范围、可能影响的其他页面。不要修改文件。

### Agent B：空白页拆解

提示词：只基于用户描述和第一张效果图分析“报告智能化空白页”。按红框拆解顶部指标、空任务引导、报告生成入口、报告审核入口、报告列表、底部流程、右侧报告类型与模板。输出字段、按钮、流程、布局比例、响应式风险。

### Agent C：报告生成页拆解

提示词：只基于用户描述和第二张效果图分析“报告生成页”。拆解顶部配置条、左侧章节目录、中间编辑预览、右侧依据来源、底部生成流程、右下生成进度、顶部操作按钮。

### Agent D：报告审核页拆解

提示词：只基于用户描述和第三张效果图分析“报告审核页”。拆解顶部标题操作、基础信息、左侧上传与规则、中间审核进度/问题统计/问题清单/预览/复核导出、右侧问题详情与处理。
